Adds a widget that shows the post from a given set of tags.

== Description ==

Tag Posts Widget is a light widget to show posts from a given set of tag.

Ispired by Category Posts Widget by James Lao - http://jameslao.com

Features:

*   Select any number of tag to be shown.
*   Add the tag browsed, while browsing the tag archives.
*   Set how many posts to show.
*   Option to append the tag names to the title.
*   Multiple widgets.

== Installation ==

1.  Download the plugin.
2.  Upload it to the plugins folder of your blog.
3.  Goto the Plugins section of the WordPress admin and activate the plugin.
4.  Goto the Widget tab of the Presentation section and configure the widget.
